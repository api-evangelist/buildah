Buildah version 1.39.0 Release Announcement We’re pleased to announce the release of Buildah version 1.39.0, which is now available from GitHub for any Linux distro. We are shipping this release on Fedora 40 and Fedora 41. Buildah will also be shipped on CentOS, OpenSUSE, and Ubuntu soon.
