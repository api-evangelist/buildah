# Buildah

Buildah is an open-source, Linux-based command-line tool for building OCI-compliant container images without requiring a full container runtime or daemon. It supports rootless builds in unprivileged environments and integrates with Podman and Skopeo as part of the containers organization. Widely used in Kubernetes-based CI/CD pipelines as a Docker-in-Docker alternative.
